# EMBL-EBI / ENA Search, Accession Number, Similar Sequence Search, and Data Download

## 1. EMBL-EBI Database Search

EMBL-EBI is a major European biological database resource used to search nucleotide sequences, protein records, gene-related data, and other biological information through its different database services.

- Biological records can be searched via several keywords (eg. gene name, organism)
- Search results may include multiple related records
- To retrieve the desired sequence, it is important to confirm every detail that matches what you need

## 2. Accession Number

An accession number is a unique identifier assigned to a biological sequence or database record.

### Importance

- helps identifying an exact record
- avoids confusion to genes that have similar names
- can be used directly for sequence retrieval or sequence similarity searches

## 3. EMBL-EBI Search for Similar Sequences

EMBL-EBI Search can be used to explore related biological records and find similar sequences using an accession number or a sequence query.

- accession number or sequence can be used as input
- helps finding related or similar sequence records
- useful for exploring sequence similarity in EMBL-EBI resources

## 4. Downloading Raw Data

Sequence data can be downloaded from ENA for later analysis.

### Common download formats

1. FASTA
2. EMBL
3. sometimes other metadata-rich formats

### Practical

1. Explore EMBL Databases:
- Go to the EMBL-EBI website (https://www.ebi.ac.uk/) and navigate to the "Databases" tab
- Explore the different databases available such as ENA, UniProt, and ArrayExpress
- Click on each database to learn more about its content and purpose

2. Retrieve an accession number from ENA:
- Go to the ENA database on the EMBL-EBI website
- Use the search bar to find a specific gene or organism you are interested in
- Click on a result to view details of the sequence
- Copy the accession number of the sequence

3. Use EMBL-EBI Search to find similar sequences:
- Go to the EMBL-EBI Search page on the EMBL-EBI website
- Enter the accession number of the sequence you retrieved from ENA or a sequence of your own
- Run the search and explore the results to find similar sequences

4. Download a Sequence from ENA:
- Go back to the ENA database and enter the accession number of the sequence found earlier
- Locate the "Download" option and choose the format in which you want to download the sequence (eg. FASTA)
